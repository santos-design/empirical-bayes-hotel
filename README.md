# Empirical Bayes Shrinkage para Análise de Cancelamentos Hoteleiros

## Visão Geral
Este projeto implementa estimação Bayesiana empírica para resolver o problema de estimar taxas de cancelamento com dados esparsos.

## Como executar
1. Clone o repositório
2. Instale as dependências no R: `install.packages(c("tidyverse", "MASS", "VGAM", "gridExtra", "knitr", "bbmle"))`
3. Baixe os dados do [Kaggle](https://www.kaggle.com/datasets/jessemostipak/hotel-booking-demand) e coloque na pasta `data/`
4. Execute `source("scripts/analyse_hotel.R")`
